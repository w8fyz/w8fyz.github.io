- SDK Java de Ship24 -

Un projet de SDK (Software Development Kit) réalisé comme projet personnel pour permettre l'utilisation de l'API (Application Programming Interface) de la plateforme Ship24 en Java. Toutes les requêtes du tier gratuit présent dans la documentation ont été mise en place et un système de de gestion d'évènements lors de mise à jour de livraison a aussi été ajouté. Le projet est désormais open source.
